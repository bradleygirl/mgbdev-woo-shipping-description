Add custom descriptions to WooCommerce shipping methods that appear on cart and checkout pages. Works with both classic and block themes.

== Description ==

WooCommerce Shipping Description adds a custom "Description" field to all shipping method settings in the WooCommerce admin. This description appears as small italic text below the shipping method name on both cart and checkout pages.

**Features:**
* Add descriptive text to any shipping method (e.g., "3-5 business days", "Ships from our warehouse")
* Works with all shipping methods (Flat Rate, Free Shipping, Local Pickup, etc.)
* Compatible with both classic and block-based cart/checkout
* Simple admin interface
* Lightweight and secure

**Use Cases:**
* Display estimated delivery times
* Add shipping policies or notes
* Clarify shipping terms and conditions
* Provide pickup instructions for local delivery

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mgbdev-woo-shipping-description` directory, or install the plugin through the WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure your shipping methods in WooCommerce > Settings > Shipping
4. Edit any shipping method and you'll see a new "Description" field at the bottom
5. Enter your description text and save

== Frequently Asked Questions ==

= Does this work with all shipping methods? =

Yes, this plugin works with all WooCommerce shipping methods including Flat Rate, Free Shipping, Local Pickup, and third-party shipping methods.

= Can I add HTML to the description? =

The description field supports limited HTML for security. Basic formatting is allowed, but scripts are stripped for security.

= Will descriptions work with both classic and block themes? =

Yes, this plugin is compatible with both classic and block-based WooCommerce cart and checkout pages.

= Do I need to know code to use this plugin? =

No, everything is done through the WordPress admin interface. No coding knowledge required.

== Privacy Policy ==

This plugin does not collect, store, or transmit any user data. It only adds a description field to shipping method settings that administrators can configure.
